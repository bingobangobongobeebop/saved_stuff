# Snippet Keeper

A lightweight local web application for organizing reusable text snippets into folders.

## Getting started

1. Clone or download this repository.
2. Open `index.html` in your preferred browser (no build tools or servers required).

All data is stored in your browser's `localStorage` so it stays on your machine.

## Features

- Create nested folders to organize snippets.
- Add, edit, rename, copy, and delete text snippets.
- Export all snippets and folders to a single JSON file for safekeeping.
- Import a previously exported file to restore your library on any machine.
